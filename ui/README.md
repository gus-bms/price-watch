# Price Watch UI (React + Vite)

This UI is a plain React application powered by Vite.
It consumes the API server from the root project (`http://localhost:4000` by default).

## Development

```bash
npm install
npm run dev
```

By default, open `http://localhost:5173`.

## API URL

Create `.env` from `.env.example` if you need a custom API host:

```bash
VITE_API_BASE_URL=http://localhost:4000
```

## Build

```bash
npm run build
npm run preview
```
